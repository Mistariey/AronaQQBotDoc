`#转账@[用户] [金钱数量]` 转账@的用户金钱

`#攻略` 查看蔚蓝档案攻略

`#塔罗牌` 抽取一张塔罗牌

`#活动` 查看BA活动

`#十连` BA卡池十发抽奖

`#单抽` BA卡池单发抽奖

`#今日运势` 查看今日的运势

`#语音生成 [角色] [内容] <中/日/英>` 蔚蓝档案角色语音生成

`#mcp ip` 查看mc服务器信息

#### <span id=fish></span>钓鱼系统

##### <span id=fishAttention></span>注意

**请注意，群聊默认没有钓鱼权限，请找 _群主_ 发送`#开启 钓鱼` 来开启钓鱼权限。同时，_群主_ 发送 `#关闭 钓鱼` 也可以关闭钓鱼权限**

##### <span id=fishStart></span>初始化
`#购买鱼竿` 来花费** 500金币 **购买一个钓竿

`#钓鱼` 在本群开始钓鱼，当阿洛娜提示 `@xxx有动静了，快来！`时执行下方指令

##### <span id=fishCommand></span>指令
注意，指令需要阿洛娜提示有动静了再发，每个指令有一定概率成功，成功会有相对应的奖励（钓到鱼的概率、钓到的鱼的品质等），最后以`#收杆`为结束钓鱼指令

`#向左拉`、`#向右拉`、`#收线`、`#放线`、`#收杆`

**当做出除`#收杆`之外的钓鱼指令时，调到正面物品的概率会大大增加。别怪我没提醒，一直掉到扣钱的物品啊！**


## <span id=Activity></span>活动
### <span id=PermOpen></span>权限开放
~~_2024/05 - 2025/01 (如果阿洛娜还在的话)_~~
::: important 由于设计不周，现已取消权限开放活动
:::

::: note 过时内容
> 阿洛娜推出船新活动！【推广机制】
> 
> 将阿洛娜推广至某一群聊（不违规使用、违反tx服务协议、不是小号），算一次有效推广，给予邀请者“升级权限组”一次奖励，请勿滥用阿洛娜哦
>
> [权限组]
> 
> 【默认】有绝大部分指令使用权限
> 
> 【VIP】推广 2 次后获得，拥有语音生成功能，若该人为某群群主或管理员，则可以使用"#监控"指令

划分出两个权限组，分别为 `default` 与 `VIP`

`VIP权限组` 需要推广2次阿洛娜，完成活跃指标后针对 **个人** 进行提权

`VIP权限组` 保留所有`default权限组`的权限，并获得新权限：

> `#监控`
> 
> `#语音生成 [角色] [文本] <中/日/英>`
> 
> `#盒 @xxx`
> 
> `#盒我`

后续还会伴随新功能加入而增加更多权限。

不仅如此，在`VIP权限组`的用户还可以 **_真正拥有_** 阿洛娜的好友（指阿洛娜会添加在`VIP权限组`的用户，做到双向好友），每次进行新功能或DEBUG测试时会随机抽取一位在该权限组内的用户联系

:::

### <span id=DebugUser></span>充电兑换
用户可以从 [爱发电](https://afdian.net/a/travellerr) 充电 **五元档位** 或 **十元档位** 来获取兑换码（由于现在我没有通过创作者认证，无法自动发送。请及时查看爱发电私信）

兑换码可在任意有阿洛娜的群聊中发送 `#认证 兑换码内容` 来兑换。兑换完成后会获得 `一个月的测试用户权限`

以下为测试版详细信息：

> 测试版拥有额外的特权
> 
> 每次指令上线前会先推送给测试版用户尝鲜
> 
> 一个月后再同步到正式版
> 
> 有些指令可能会作为VIP用户内容，但上线前仍会推送给测试版用户
> 
> 一个月内没有测试内容，则权限再次延长一个月！
